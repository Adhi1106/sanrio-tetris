## Title
Sanrio Tetris Game

Play here!: https://adhi1106.github.io/sanrio-tetris/
## Description
A browser-based Tetris game inspired by Sanrio aesthetics. Built using HTML, CSS, and JavaScript with a focus on game logic, DOM manipulation, and interactive UI design.

## Features
- Classic Tetris gameplay logic
- Keyboard-based controls
- Score tracking
- Custom Sanrio-themed UI
- Responsive layout for browser play

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Structure
index.html – main game layout

style.css – game styling and theme

script.js – game logic and interactions

assets/ – images and UI assets

README.md - project documentation

## What I Learned
- DOM manipulation using JavaScript
- Game loops and state management
- Handling keyboard events
- Structuring a front-end project
